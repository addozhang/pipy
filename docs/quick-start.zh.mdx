---
title: "快速入门"
---

从 [Github](https://github.com/flomesh-io/pipy) 克隆代码。

```shell
git clone https://github.com/flomesh-io/pipy.git
```

在 `tutorial` 目录中有脚本的开发教程，这里使用 `tutorial/01-hello/hello.js` 作为示例，在该脚本中，监听 `8080` 端口的服务会返回请求的内容。

## 二进制文件

在 Pipy 仓库的[发布页面](https://github.com/flomesh-io/pipy/releases/latest)，下载对应平台的二进制文件，并解压。

```shell
pipy tutorial/01-hello/hello.js

curl -x POST http://localhost:8080 -d 'hello world'
hello world
```

## Docker

```shell
docker run --rm --entrypoint "pipy" -v ${PWD}/tutorial/01-hello:/script -p 8080:8080 flomesh/pipy-pjs:latest /script/hello.js

curl -x POST http://localhost:8080 -d 'hello world'
hello world
```
