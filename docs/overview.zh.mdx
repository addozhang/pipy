## 介绍

[Pipy](https://github.com/flomesh-io/pipy) 是一个高性能的可编程网络代理。Pipy 是用 C++ 编写的，非常轻量级和快速，使其成为服务网格代理的理想解决方案。

借助内置的 JavaScript 支持，得益于 PipyJS，Pipy 具有高度可定制性和可预测性，因为 PipyJS 选择了引用计数内存管理，因此没有我们在其他类似系统中经常看到的不均匀垃圾收集开销。

在其核心，Pipy 是一个模块化设计，提供一系列基本过滤器，可以链接起来形成一个管道，网络数据被泵入并获取处理。即使在服务网格环境之外，Pipy 的设计方式也使其非常通用。由于其可编写脚本的性质，它还有益于涉及网络节点之间的中间消息处理的任何类型的情况，其中 Pipy 可用作前向/后向代理、负载均衡器、API 网关、静态 HTTP 服务器、CDN 中的节点、健康检查器、无服务器函数触发器，或者您想通过几行脚本组合所有这些功能。

## 兼容性

Pypi 旨在实现跨不同操作系统和 CPU 架构的高兼容性。 Pypi 已经在这些平台上进行了全面测试：

* CentOS 7
* Ubuntu 18/20
* FreeBSD 12/13
* macOS Big Sur

生产环境推荐使用 CentOS7/REHL7 或 FreeBSD。
