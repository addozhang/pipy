## Description

Error type of the **StreamEnd** event.

When the **StreamEnd** event is due to a normal pipeline closing up without an error, this property will be an empty string. Otherwise, it is one of the following strings indicating what error has happened.

- "UnknownError"
- "RuntimeError"
- "ReadError"
- "WriteError"
- "CannotResolve"
- "ConnectionCanceled"
- "ConnectionReset"
- "ConnectionRefused"
- "ConnectionTimeout"
- "ReadTimeout"
- "WriteTimeout"
- "Unauthorized"
- "BufferOverflow"

## Syntax

``` js
streamEnd.error
```
