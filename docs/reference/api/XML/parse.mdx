## Description

Parses text in a string as XML document and returns its root node.

The input to this method is expected to be a string. If you have a [Data](/reference/api/Data) input, use [XML.decode()](/reference/api/XML/decode) instead.

## Syntax

``` js
XML.parse(text)
```
