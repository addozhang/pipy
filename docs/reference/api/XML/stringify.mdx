## Description

Formats an XML document in text.

This method returns a string for the formatted text. When only the binary data is needed, use [XML.decode()](/reference/api/XML/decode) instead, which can be more efficient due to the absence of an extra string conversion.

## Syntax

``` js
XML.stringify(node)
```
