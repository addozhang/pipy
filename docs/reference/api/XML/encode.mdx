## Description

Formats an XML document in text.

This method returns a [Data](/reference/api/Data) for the formatted text. When a string is needed, use [XML.stringify()](/reference/api/XML/stringify) instead, which could be a bit lower in performance.

## Syntax

``` js
XML.encode(node)
```
