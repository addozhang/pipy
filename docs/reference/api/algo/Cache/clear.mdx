## Description

Deletes all entries in **Cache**.

If a _remove_ callback was given while constructing the **Cache** by [new Cache()](/reference/api/algo/Cache/constructor), it will be called with every key-value pair while they are deleted.

## Syntax

``` js
cache.clear()
```
