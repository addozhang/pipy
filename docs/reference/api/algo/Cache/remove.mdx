## Description

Deletes an entry in **Cache**.

If a _remove_ callback was given while constructing the **Cache** by [new Cache()](/reference/api/algo/Cache/constructor), it will be called with the key-value pair while it is deleted.

## Syntax

``` js
cache.remove(key)
```
